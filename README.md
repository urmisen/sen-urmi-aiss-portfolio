# AISS Portfolio Website

Portfolio website for AISS (Advanced Innovation in Sustainable Societies) scholarship application, showcasing 5 interactive projects demonstrating AI/ML expertise for sustainable social innovation.

## Overview

This interactive portfolio website presents five key projects that highlight technical skills, contributions, and their connection to sustainable societies. Each project features an interactive slideshow with five distinct sections: Background & Problem Statement, My Contribution, Skill Set Demonstrated, Connection to Sustainable Societies, and Resources.

## Features

- Interactive project slideshows with navigation via indicators, keyboard arrows, and touch/swipe gestures
- Responsive design optimized for desktop, tablet, and mobile devices
- Light mode theme with modern, clean aesthetics
- Dynamic vertical centering for optimal viewing experience
- Categorized skill visualization with card-based layouts
- Two-column layouts for project images and content
- Smooth scrolling navigation

## Projects Featured

1. **Intelligent Regulatory Compliance Assistant** - RAG-based knowledge management system for financial compliance
2. **Dynamic Pricing & Operational Zoning System** - Geospatial optimization for urban mobility
3. **Causal AI for Optimizing Driver Incentives** - Causal inference framework using Generalized Random Forest
4. **DeepGlut - Deep Learning Framework for Protein Glutarylation Site Prediction** - CNN-based bioinformatics solution
5. **QSiamNet - Hybrid Quantum-Classical Siamese Network** - Quantum machine learning for efficient similarity learning

## Getting Started

### Local Development

1. Clone the repository:
```bash
git clone https://github.com/urmisen/sen-urmi-aiss-portfolio.git
cd sen-urmi-aiss-portfolio
```

2. Serve the files using a local web server. You can use one of the following methods:

**Using Python:**
```bash
python3 -m http.server 8000
```

**Using Node.js (http-server):**
```bash
npx http-server -p 8000
```

3. Open your browser and navigate to:
```
http://localhost:8000
```

### GitHub Pages

The website can be hosted on GitHub Pages:

1. Go to repository Settings
2. Navigate to Pages section
3. Select `main` branch as source
4. The site will be available at: `https://urmisen.github.io/sen-urmi-aiss-portfolio/`

## Navigation

- **About**: Personal introduction and professional journey
- **Portfolio**: Interactive project slideshows
- **Skills**: Categorized technical skills
- **Contact**: Contact information and social links

## Slideshow Navigation

Each project slideshow can be navigated using:
- Slide indicators (dots at the bottom)
- Keyboard arrow keys (Left/Right)
- Touch/swipe gestures on mobile devices
- Click on slide area (left half for previous, right half for next)
